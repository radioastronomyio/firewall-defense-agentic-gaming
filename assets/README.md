## 3. Style Guide

All images follow a consistent visual style:

- Blueprint grid background
- Cream/beige content panels
- Flat iconography
- Muted color palette
- Flow arrows for process visualization
- Concise labels

Images are generated using Google AI Studio (Gemini) with style consistency maintained across the repository ecosystem.
